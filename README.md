An innovative personal portfolio reflecting the intersection of traditional software development and the new era of Artificial Intelligence.

## Tech Stack
- **Framework:** Next.js 14 (App Router, TypeScript)
- **Styling:** Tailwind CSS
- **Components:** Shadcn/UI
- **Animations:** Framer Motion
- **Deployment:** GitHub Pages

## Features
- **AI Interface Design:** "Prompt Bar" hero section and neon aesthetics.
- **Internationalization:** Full English/Spanish support.
- **Interactive Elements:** Scroll animations, glow effects, and micro-interactions.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

This project is configured for deployment on GitHub Pages.

```bash
npm run deploy
```
